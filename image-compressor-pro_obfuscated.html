Frequently Asked Questions

Is my data safe?

Yes! All image processing happens locally in your browser. Your images are never uploaded to our servers.

What file formats are supported?

We support JPEG, PNG, WebP, GIF, and BMP formats for both input and output.

What's the maximum file size?

Each image can be up to 10MB in size. For batch processing, you can upload multiple files.

How does the compression work?

We use advanced algorithms to reduce file size while maintaining visual quality. You can adjust the quality slider to find the perfect balance.
